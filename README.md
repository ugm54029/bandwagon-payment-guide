## 四种支付方式详细说明

### 支付宝

扫码付款，最省事。进入付款页后点「Pay now」，等一下会出现支付宝二维码，打开支付宝扫一下，手机上确认付款，搞定。

**一个坑**：不要挂代理挂着 VPN 去下单付款，这种操作很容易被标记成欺诈订单，轻则支付失败，重则账号被封。搬瓦工的风控对代理 IP 很敏感。

### 银联

走 Payssion 这个第三方支付网关，本质上是银联的接口，安全没问题。输入卡号后进入银联页面，选「直接支付」或「网银支付」：

- 直接支付：在银联页面完成，输卡号、验证
- 网银支付：跳到你银行自己的页面，有些银行还能用手机 App 扫码，挺方便

国内银行卡基本都是银联卡，随便哪张都行。

### PayPal

需要有 PayPal 账户。结算页面选 PayPal，跳到贝宝那边完成授权付款。PayPal 偶尔有接口问题，遇到付款失败的，换支付宝或银联通常能过。

### 信用卡

仅支持 Visa 和 Mastercard。国内的双币卡或者全币卡可以用，但要确认开通了境外支付功能。

## 支付宝付款失败怎么办

这个情况不算少见。支付宝跳转后页面空白、提示「系统有点儿忙，一会儿再试试，错误码：SECONDARY_MERCHANT_STATUS_ERROR」，或者扫码后一直卡着没反应。

碰到这种情况，步骤是这样的：

1. **先等几分钟再试一次**，有时候只是网络抖动
2. **切换到银联支付**：回到付款页，找「Change Payment Method」，换成银联重新发起
3. **如果订单已经生成但没付款**：登录账户后台，在首页能看到未支付账单，点「View Invoice」然后重新选支付方式付款就行，不用重新走注册流程

注意：付款失败了不等于订单消失，账单还在的，登录进去找到再付就行。

## 充值余额和正常购买有什么区别

这是新手容易搞混的一点。

**正常购买（首次买套餐或续费账单）**：支持支付宝、银联、PayPal、信用卡，四种都可以。

**充值账户余额（Add Funds）**：只支持 PayPal 和信用卡，不支持支付宝和银联。而且充值后不可退款，所以没有必要专门充值，等收到续费账单邮件再付就行。

多数人用不上充值余额这个功能，直接正常付款就够了。
